LIAM'S PECS LIBRARY
===================

Built from the PECS sheets you shared. Every unique picture card was extracted,
cleaned up, de-duplicated and sorted into categories. 294 unique cards in total.

HOW IT IS ORGANISED
-------------------
PECS_Library/
  index.html            <- open this in any web browser to browse everything
  <Category>/           <- one folder per category
     <name> - full.png
     <name> - no border.png
     <name> - no label.png
     <name> - photo only.png
     <name> - clean.png

THE 5 VERSIONS OF EACH CARD
---------------------------
- full        The card as it is now: black rounded border, picture and printed label.
- no border   Frame removed. Rounded card on a transparent background.
- no label    Border and picture kept, the wording removed.
- photo only  Just the picture, no frame and no words.
- clean       Rebuilt card: white rounded card, black border, picture, and the label
              re-typed in crisp sharp text (best for printing).

All images are PNG. The bordered/rounded versions have transparent backgrounds
around the card so they layer cleanly onto any design.

NOTES
-----
- Some things have more than one picture (e.g. several alpaca photos). These are kept
  as 'alpaca 1', 'alpaca 2' etc. so you have options.
- Labels were read from the printed cards. A few odd ones (e.g. cards printed with a
  stock-photo number) were given a sensible descriptive name — rename any file freely.
- Everything is yours to edit, print and reuse in future projects.
