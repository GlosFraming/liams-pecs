# Liam's PECS

A small offline-capable website for Liam's picture communication (PECS):
a home hub with a **Card Library**, a **PEC Maker** (add your own cards), and a
**Print Sheet** (print cards to cut out, default 5 cm).

Open `index.html` to run it. It is a plain static site — no build step needed.

> Private / personal: contains photos of family and carers. Keep this repository
> private and restrict the published site (e.g. with Cloudflare Access).
